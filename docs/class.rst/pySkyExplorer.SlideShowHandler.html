setLoad((SlideShowHandler)arg1, (object)load) → None :

PARAMETERS: load : str Load an XML file representing a slide show. Path must be relative to user folder.

setPause((SlideShowHandler)arg1, (bool)pause) → None :

PARAMETERS: pause : bool Pause current playing slide show. If no slide show currently playing, it has no effect.

setStop((SlideShowHandler)arg1, (int)stop) → None :

PARAMETERS: stop : int Stop current playing slide show. If no slide show currently playing, it has no effect.

setUserAction((SlideShowHandler)arg1, (UserAction)userAction) → None :

PARAMETERS: userAction : UserAction Trigger a user action, see ‘UserAction’ documentation for available values.

property beginningOfSlideShow
