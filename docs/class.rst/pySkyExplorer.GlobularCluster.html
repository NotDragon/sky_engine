[Read-only] Position of the globular cluster in the ICRF coordinate system.

[Read-only] Radius ratio of the globular cluster according to the unit of it’s coordinate system. In most case, value is globular cluster radius in kilometers.

Scale factor of the globular cluster. It can be used to enlarge apparent size of the globular cluster.
