[Read-only] Position of the nebula in the ICRF coordinate system.

[Read-only] Radius ratio of the nebula according to the unit of it’s coordinate system. In most case, value is nebula radius in kilometers.

Scale factor of the nebula. It can be used to enlarge apparent size of the nebula.
