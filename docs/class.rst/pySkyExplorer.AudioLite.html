pySkyExplorer.AudioLite

class pySkyExplorer.AudioLite
load((AudioLite)arg1, (object)filename) → None :

Load an audio file. PARAMETERS: filename : str

Path to the audio file to read.

Can be a relative path to audio folder or an absolute path.

pause((AudioLite)arg1) → None :

Pause the current playing audio file.

play((AudioLite)arg1, (Animator)animator) → None :

Play the current playing audio file. PARAMETERS: animator : Animator default value : Animator()

seek((AudioLite)arg1, (float)time) → None :

Change the current audio timing. PARAMETERS: time : double

Position to reach in the audio file (in seconds).

setVolume((AudioLite)arg1, (float)volume) → None :

Set the volume of the current loaded audio file. PARAMETERS: volume : double

New volume for loaded audio file.

stop((AudioLite)arg1) → None :

Stop the current playing audio file.
